# my_shell

## Installation

Clone this repository and compile as follow : 
```bash
gcc -Wall  -Wextra -pedantic -std=c99 *.c -o my_shell
```

## Usage

Execution : 
```bash
./my_shell
```

# Examples

```bash
my_shell> ls
builtin.c  main.c    my_shell.c  new_process.c	reading.c
exec.c	   my_shell  my_shell.h  parsing.c
```
